Timothy John Berners-Lee nasceu em Londres, Inglaterra, em 8 de junho de 1955.
Ele é um físico britânico, cientista da computação e professor do MIT (Instituto de Tecnologia
de Massachusetts, em inglês). Seu cargo mais conhecido é ser o criador da World Wide Web, o famoso termo técnico WWW.

Em 1989, Berners-Lee viu a oportunidade de unir hipertexto com internet e criar a World Wide Web. O primeiro site foi construído no CERN e foi ao ar em 1991
e era uma página simples de texto que oferecia uma explicação sobre o que era a World Wide Web, como alguém poderia criar um navegador, configurar um servidor web, etc.
Em 1994, fundou o World Wide Web Consortium (W3C) no MIT. É composto por várias empresas que estavam dispostas a criar normas e recomendações para melhorar a qualidade na web.


Popularização HTMLVoltar

Era tudo muito rudimentar e bem diferente dos sites que temos hoje a nossa disposição, mas consistia de um início e que permitia criar mais do que uma simples página de texto corrido, como se tinha nos livros, por exemplo. E com isso você automaticamente já sabe que foi a linguagem que permitiu o desenvolvimento dos primeiros sites. E se por um lado, muita coisa mudou, por outro saiba que algumas não mudaram e todo site que você acessa hoje, ainda usa o HTML, em maior ou menor grau.

É verdade que desde então o HTML evoluiu muito. A popularização da linguagem teve como ponto de partida os conceitos de Berners-Lee, lançando em 1995 a versão 2.0 do HTML, para ao longo de quatro anos, sofrer mudanças e implementações que permitiram que em 1999 a versão mais usada até hoje – a 4.1 – fosse lançada pelo W3C, que é a comunidade que normatiza e desenvolve padrões para como devem ser as coisas na Internet.

Tags HTMLVoltar

Arthur Paredes Iniciar no mundo da programação web é uma decisão que cada vez mais pessoas tomam, conscientes das oportunidades profissionais envolvidas em ter esse tipo de conhecimento tão valorizado pelas empresas atuais. Entender o significado e a composição da tag HTML é um dos primeiros passos que qualquer profissional que queira se tornar um desenvolvedor ou engenheiro web deve dar.

Para que você entenda o que é uma tag HTML, queremos ensiná-lo como funciona a camada mais básica de um site e que você pode se lançar para projetar o seu sem medo. Você está conosco neste guia introdutório?

HTML é uma markup language, o que significa que ela é escrita com códigos que uma pessoa pode ler sem que seja necessário compilá-la primeiro. Em outras palavras, o texto em uma página da Web é “marcado” com esses códigos para instruir o navegador da Web sobre como exibir o texto. Essas tags de marcação são as próprias tags HTML.

Quando você escreve código em HTML, você está escrevendo tag HTML. Todas as tags HTML são feitas com uma série de partes específicas, incluindo:

  1. O personagem “menos que” <;
    Uma palavra ou caractere que determina qual rótulo está sendo digitado;

  2. Qualquer número de atributos HTML que você deseja usar,
    escrito no nome do formulário=”valor”;

  3. O personagem “maior que” >.

HyperText (HTML) é uma linguagem de computador que forma a maioria das páginas da Web e plataformas online. HTML não é considerado uma linguagem de programação porque não pode criar funcionalidade dinâmica. No entanto, os usuários da Web podem criar e estruturar seções, parágrafos e links usando elementos, tags e atributos.

Atualmente, existem 142 tags HTML disponíveis que permitem a criação de vários elementos. Porém de 142,Usamos pouco menos de 100, pois alguns novos computadores não suportem mais alguns desses, é importante estar ciente da existência deles.
